# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, available at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

By participating in this project (issues, pull requests, discussions, or any other interaction in this repository), you agree to abide by its terms.

## Reporting

Instances of unacceptable behavior may be reported to the project maintainers at <pc@pandoracloud.net>. Reports are reviewed and investigated promptly and fairly. Reporter privacy and safety are prioritized.

For security-related reports (vulnerabilities, credential exposure), see [SECURITY.md](SECURITY.md) instead.

## Scope

This Code of Conduct applies within all project spaces, and also applies when an individual is officially representing the project in public spaces.
